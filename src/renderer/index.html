üí° What is Work Launcher?

Work Launcher is a powerful desktop application designed to streamline your daily workflow by automating the process of opening multiple applications at once. Whether you're starting your workday, switching between different projects, or managing various work contexts, Work Launcher eliminates the repetitive task of manually opening each application.

üìã Profiles

Create different profiles for different work contexts. Each profile saves which apps are enabled.

  • Create Profile: Click the + button next to profiles dropdown
  • Switch Profile: Select from the dropdown to load different app configurations
  • Edit Profile: Click the pencil icon ‚úèÔ∏è to rename a profile
  • Delete Profile: Click the trash icon üóëÔ∏è to remove (cannot delete "Default")
  • Save to Profile: After enabling/disabling apps, they auto-save to current profile

Example profiles: "Development" (IDE, terminal, browser), "Meetings" (Teams, Outlook), "Design" (Figma, Photoshop)

‚ûï Adding Apps

Click "+ Add App" to open the Add Application dialog:

  • Installed Apps Tab: Shows apps detected from Start Menu. Click to add.
  • Browse Tab: Click "Browse" to select any .exe, .bat, or .cmd file
  • URL Tab: Add websites/web apps that open in your default browser
  • Custom Name: Enter a display name for the app
  • Arguments: Add command-line arguments if needed (e.g., --profile="Work")

üéõÔ∏è Managing Apps

  • Enable/Disable: Toggle the switch to include/exclude app from launch
  • Reorder: Drag the ‚ãÆ‚ãÆ handle to change launch order
  • Edit: Click the pencil icon to change name or path
  • Remove: Click the X button to delete app from list
  • Chrome Profiles: If Chrome is added, select which browser profile to use

‚ö° Launching Apps

  • Launch All: Click "üöÄ Launch All" to start all enabled apps in order
  • Launch Delay: Set delay between app launches (0-5 seconds) to prevent system overload
  • Startup Dialog: When enabled, shows confirmation before launching on Windows startup
  • Quick Launch: Right-click tray icon ‚Üí Quick Launch to start individual apps

üîë How to Get Jira API Token

Step-by-step guide:

  1. Enter Server URL First

    Enter your Jira server URL in the settings. Once added, you'll see a direct "Create one here" link below the API token field.

  2. Click the Token Link

    Click the "Create one here" link that appears, or manually visit your Jira profile settings

  3. Navigate to Security/API Tokens

    Look for "Security" tab or "Personal Access Tokens" section in your profile

  4. Create API Token

    Click "Create API token" or "Create token" button. Give it a name like "Work Launcher"

  5. Copy and Paste

    Copy the token immediately (you won't see it again!) and paste it in the API Token field

üí° Tip: For Jira Cloud (atlassian.net), use your Atlassian account email. For self-hosted Jira Server/Data Center, use your Jira username.
